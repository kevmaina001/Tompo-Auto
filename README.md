## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies: npm install
3. Set up environment variables in .env.local
4. Start Convex: npx convex dev
5. Start Next.js: npm run dev
